# Kali

# nmap [掃描方式] [命令選項] {目標}

 掃描目標格式：
```
IPv4地址： 192.168.1.1
IPv6地址：AABB:CCDD::FF%eth0
主機名：www.target.tgt
IP地址範圍：192.168.0-255.0-255
掩碼格式：192.168.0.0/16
使用列表文件：- iL <filename>
```

設置掃描端口
```
無端口範圍時掃描1000 個常用端口

-F掃描100個最常用端口
-p<port1>-<port2>指定端口範圍
-p<port1>,<port2>,….端口列表
```
